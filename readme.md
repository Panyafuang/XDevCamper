# XDevCamper API

> Backend API for XDevCamper application, wich is a bootcamp directory website

## Usage
Rename "config/config.env.env" to "config/config.env" and update the value/setting to your own

# Install Dependencies
````````````
npm install
````````````

# Run app
````````````
# Run in dev mode
npm run dev

# Run in prod mode
npm start
````````````

- Version: 1.0.0
